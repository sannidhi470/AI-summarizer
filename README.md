# AI-summarizer

This project is a simple REST API built using Spring Boot that takes a block of text as input and uses an AI model to generate a concise summary along with three key action items.

The application integrates with the OpenAI API to process input text and return results in a structured JSON format. It follows a clean architecture by separating concerns into controller, service, and model layers.

Special focus was placed on prompt design to ensure consistent, reliable, and well-structured AI responses.
